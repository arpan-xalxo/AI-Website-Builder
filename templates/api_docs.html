AI Website Builder API

Complete API documentation for AI-driven website generation and management

Base URL: http://localhost:5000

Overview

This API provides a complete solution for AI-driven website generation and management with role-based access control.

Features:

  • JWT-based authentication
  • Role-based access control (Admin, Editor, Viewer)
  • AI-powered website generation (Gemini)
  • Website CRUD operations
  • Live website preview
  • Rate limiting and caching

Authentication

All authenticated endpoints require a JWT token in the Authorization header.

POST

Sign Up

/signup

Register a new user account.

Request Body:
{
  "email": "user@example.com",
  "password": "password123",
  "role": "Editor"
}
Response:
{
  "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}
POST

Login

/login

Authenticate user and get JWT token.

Request Body:
{
  "email": "user@example.com",
  "password": "password123"
}

Admin APIs

Role and user management (Admin only).

GET

List Roles

/roles

Get all available roles.

Headers:
Authorization: Bearer <jwt_token>
POST

Create Role

/roles

Create a new role.

Request Body:
{
  "name": "Moderator",
  "permissions": ["view_websites", "edit_own_website"]
}

Website APIs

Website CRUD operations with role-based permissions.

POST

Create Website

/websites

Create a new website (Admin/Editor).

Request Body:
{
  "data": {
    "title": "My Website",
    "hero_section": {
      "heading": "Welcome",
      "subheading": "Subtitle"
    }
  }
}
GET

Get Websites

/websites

Get all websites (filtered by role).

GET

Get Website

/websites/<website_id>

Get specific website by ID.

AI Generation

Generate website content using AI models.

POST

Generate Website

/generate-website

Generate website content using AI (Gemini).

Request Body:
{
  "business_type": "Restaurant",
  "industry": "Food & Beverage",
  "description": "Italian restaurant",
  "model": "gemini"
}
Response:
{
  "msg": "Website generated successfully",
  "website_id": "generated_website_id",
  "content": "generated_content",
  "ai_model_used": "gemini"
}

Preview

Website preview functionality.

GET

Preview Website (Public)

/preview/<website_id>

View website preview (no authentication required).

GET

Preview Website (Authenticated)

/preview-auth/<website_id>

View website preview with authentication.

API health monitoring.

Response:
